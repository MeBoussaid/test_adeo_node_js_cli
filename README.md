# Node.js CLI tool for test :

This a test for ADEO, creating a CLI tool with Node.js

# Install the project :

```bash

npm install

```

# To run unit tests :

```bash

npm test

```

# To get the data with counts :

```bash

node app.js --count

```

# To search by pattern:

```bash

node app.js --filter=<pattern>

```
